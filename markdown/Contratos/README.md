# Índice de Documentación del Sistema de Contratos

Esta carpeta contiene la documentación completa del sistema de gestión de contratos. A continuación, se presenta un índice de los documentos disponibles:

## Documentos Principales

1. [**Documentación General**](./Documentación.md) - Visión general del sistema de gestión de contratos, componentes principales y arquitectura.

2. [**Flujo de Trabajo**](./FlujoTrabajo.md) - Explicación detallada del ciclo de vida de un contrato, desde su creación hasta su finalización, incluyendo todos los eventos y notificaciones.

## Comandos Artisan

3. [**Comando de Verificación de Contratos**](./ComandoVerificarContratos.md) - Documentación del comando `contratos:verificar`, que identifica contratos vencidos y próximos a vencer.

4. [**Comando de Creación de Contratos de Prueba**](./ComandoCrearContratosPrueba.md) - Documentación del comando `contratos:crear-pruebas`, que facilita la creación de contratos con fechas de vencimiento específicas para pruebas.

## Guías Prácticas

5. **Guía de Configuración** - Instrucciones para configurar correctamente el sistema de contratos, incluyendo cron jobs y notificaciones.

6. **Guía de Solución de Problemas** - Problemas comunes y sus soluciones en el sistema de gestión de contratos.

## Anexos

7. **Diagrama de Base de Datos** - Estructura de las tablas relacionadas con contratos y sus relaciones.

8. **Glosario de Términos** - Definición de términos específicos utilizados en el contexto de contratos.
